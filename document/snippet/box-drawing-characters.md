# Box-drawing characters

## Example

```
┌─┬──┐
│ │  │
├─┴──┤
│    │
├─┬──┤
└─┴──┘
```

## Light lines

```
─  horizontal
│  vertical
┌  top-left corner
┐  top-right corner
└  bottom-left corner
┘  bottom-right corner
├  left tee
┤  right tee
┬  top tee
┴  bottom tee
┼  cross
```

## Heavy lines

```
━  horizontal
┃  vertical
┏  top-left corner
┓  top-right corner
┗  bottom-left corner
┛  bottom-right corner
┣  left tee
┫  right tee
┳  top tee
┻  bottom tee
╋  cross
```

## Double lines

```
═  horizontal
║  vertical
╔  top-left corner
╗  top-right corner
╚  bottom-left corner
╝  bottom-right corner
╠  left tee
╣  right tee
╦  top tee
╩  bottom tee
╬  cross
```

## Rounded corners

```
╭  top-left
╮  top-right
╰  bottom-left
╯  bottom-right
```

## Dashed lines

```
┄  light triple dash horizontal
┆  light triple dash vertical
┈  light quadruple dash horizontal
┊  light quadruple dash vertical
╌  light double dash horizontal
╎  light double dash vertical
```
