# Box-drawing characters

Unicode box-drawing characters for diagrams, tables, and TUI layouts.

## Templates

### Light

```
┌───┬────┐
│   │    │
├───┼────┤
│   │    │
├───┼────┤
│   │    │
└───┴────┘
```

### Heavy

```
┏━━━┳━━━━┓
┃   ┃    ┃
┣━━━╋━━━━┫
┃   ┃    ┃
┣━━━╋━━━━┫
┃   ┃    ┃
┗━━━┻━━━━┛
```

### Double

```
╔═══╦════╗
║   ║    ║
╠═══╬════╣
║   ║    ║
╠═══╬════╣
║   ║    ║
╚═══╩════╝
```

### Rounded

```
╭────────╮
│        │
│        │
╰────────╯
```

### Dashed

```
┌┄┄┄┄┄┄┄┄┐
┆        ┆
┆        ┆
└┄┄┄┄┄┄┄┄┘
```
